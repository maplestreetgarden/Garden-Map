# Maple Street Community Garden Tracker

A web app for tracking plants, objects, and activity across the Maple Street Community Garden. Built as a single HTML file, hosted on GitHub Pages, with a Supabase backend for shared data and photo storage.

## What it does

- **Interactive garden map** — upload your garden map and draw clickable zones over it
- **Items** — track plants (crop, variety, sow/harvest dates) and objects (tools, fixtures, equipment) in each zone
- **Logs** — dated entries on each item with notes, photos, custom fields, tags, and season tracking
- **Shared data** — all gardeners see the same data in near real-time across any device
- **Activity history** — admin can review a full log of who added or changed what and when
- **Photo uploads** — supports standard images and iPhone HEIC photos
